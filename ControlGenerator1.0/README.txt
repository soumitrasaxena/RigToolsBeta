ControlGenerator 1.0
------------------------

Quicken your rigging workflow by this simple script. Just select the type of control , click , and viola ! The script generates the relevant control for the handle.

How It Works :
---------------------

1. Select the type/shape of control you want for the handle/joint.
2. Modify the control accordingly.
3. Click 'Done' to finalise it , and move to the next joint.
4. Parent and animate !

Coded in Python for Autodesk Maya. Tested on Maya 2012.
